# Changelog

## [4.1.1] - 2026-01-16
- Fixed CSS gradient display issues on all browsers (forced background-clip).
- Resolved "Permission Denied" error when posting updates (Firebase Rules adjustment).

## [4.1.0] - 2026-01-16
- Restored original gradient header design concept.
- Changed Admin login to use a secret URL parameter.
- Added specific "Admin Mode" subtitle.
- Renamed navigation button to "Home Page" (דף הבית).
- Fixed Updates Feed empty state and added "Delete Post" button.

## [4.0.0] - 2026-01-16
- Major architecture change to Single Page Application (SPA).
- Added Analytics section with interactive Charts (Comparison & Individual).
- Added Updates Feed system with Rich Text support.
- Added Smart Rename feature (keeps history).
- Implemented Admin Authentication system.

## [3.0.0] - 2026-01-01
- Migrated database to Firebase Firestore (Cloud Sync).
- Real-time updates across devices.
- Basic security implementation.

## [2.0.1] - 2025-12-21
- Sparkles emoji ✨ removed from text. it was very annoying before!

## [2.0.0] - 2025-12-21
- New design
- Automatic sorting of names by points
- Delete names feature

## [1.0.0] - 2025-12-09
- Initial Release: First launch of "Liri Points".
- Local Storage: Data saved locally on the browser (no cloud sync).
- Core Features: Ability to add friends, increment/decrement points, and manual edit via prompt.
- Legacy Design: Original beige/cream color theme with "Alef" font.
